零流量接口治理与级联删除平台

鲁班内控平台技术专项 · 面试准备资料

项目概览

这是一个平台化代码治理工具，通过静态调用图分析 + 级联删除算法， 将原本需要2小时的人工治理降至15分钟，效率提升87.5%。
打通"FAST流量监控 → 调用链分析 → 自动删除 → 编译校验 → Git推送"全链路。

技术栈

核心亮点（面试重点）

  • 级联删除算法：基于图的DFS算法，递归分析可删除方法集合，处理循环依赖、共享依赖等复杂场景
  • 多重安全保障：黑名单保护 + 编译校验 + 测试流水线，误删率0%
  • 平台化设计：从人工排查升级到Web UI操作，支持批量治理、预览报告、审计追溯
  • Vibe Coding实践：AI协同开发，开发效率提升60%，25天工作量压缩到10天
  • 全链路自动化：零流量发现 → 静态分析 → 代码删除 → 编译校验 → MR创建 → CI/CD验证
